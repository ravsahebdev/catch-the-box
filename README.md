# 🎮 Catch The Box

A fast-paced and interactive browser game built with **HTML, CSS, and Vanilla JavaScript**.

Catch The Box challenges players to click a randomly moving box within a limited time. Each successful click increases the score, while cumulative points are tracked across multiple rounds with performance-based feedback.

---

## ✨ Features

- 🎯 Random Box Movement
- ⏱️ 10-Second Game Rounds
- 🏆 Real-Time Score Tracking
- 📊 Cumulative Total Points
- 🔊 Interactive Click Sound Effects
- 🎭 Performance-Based Emoji Feedback
- 🔄 Play Again Functionality
- 🎮 Dynamic Game Over Screen
- 📱 Responsive Design
- 🖱️ Desktop and Mobile Support

---

## 🎮 How to Play

1. Click the **Start** button to begin the game.
2. A box will appear at random positions inside the playground.
3. Click the box before it moves to another position.
4. Each successful click increases your score.
5. The game ends automatically after **10 seconds**.
6. Your round score and cumulative total points are displayed.
7. Click **Play Again** to start another round.

## 🚀 Running Locally

1. Clone the repository.
2. Open the project folder.
3. Launch `index.html` in a modern browser.
4. Click **Start** and begin playing.

No package installation or build process is required.

## 🎯 Project Overview

Catch The Box demonstrates how core browser APIs and JavaScript fundamentals can be combined to build a complete interactive game experience without relying on external libraries or frameworks.

The project focuses on real-time interaction, game state control, responsive UI behavior, timed events, dynamic DOM updates, and user feedback.
